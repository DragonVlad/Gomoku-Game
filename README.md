# Gomoku Game on JS

В этом проекте я создал игру Гомоку. В ней игрок играет против компьютера. Нужно поставить 5 точек в ряд прежде чем это сделает противник. Игрок играет за черных. 

Я использовал самовызывающиеся функции, объекты, события, циклы, DOM. Использовался объектно-ориентированный подход.
