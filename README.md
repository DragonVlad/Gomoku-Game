# Gomoku Game on JS

В этом проекте я создал игру Гомоку. В ней игрок играет против компьютера. Нужно поставить 5 точек в ряд прежде чем это сделает противник. Игрок играет за черных. 

Я использовал самовызывающиеся функции, объекты, события, циклы, DOM. Использовался объектно-ориентированный подход. 

## Деплой на Github Pages

1. Открываем репозиторий.
2. Переходим на вкладку Actions.
3. Жмем кнопку New workflow.
4. Вбиваем в поиск Static HTML.
5. Жмем на нем кнопку Configure.
6. Жмем справа наверху кнопку Commit changes.
7. Идем на вкладку Settings -> Pages.
8. В разделе Build and deployment находим пункт Source.
9. Выбираем Github Actions.
10. На вкладке Actions ждем пока не появится зеленая галочка.
11. Возвращаемся на вкладку Settings -> Pages, увидим ссылку на полученный сайт.
12. Копируем ее, идем на вкладку Code, в разделе About вставляем ее в поле URL.
